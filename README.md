# OpenGL Collecting Coin

This project is an **OpenGL 3D demo** that renders 3D model
---

## Features
- 3 Model render for player, coin, and ground
- Randomly generated coin fall down 
- Camera controls (WASD)

---

## Controls
- **W / A / S / D** → Move camera  
- **ESC** → Quit program
